# tmux quickstart

## Prefix
Primary: <kbd>Ctrl</kbd>+<kbd>a</kbd> — Secondary: <kbd>Ctrl</kbd>+<kbd>b</kbd>

> In all binds below, `prefix` means hit the prefix first, then the key.

---

## Terminal commands

| Command | Action |
|---|---|
| `tmux` | New session |
| `tmux new -s name` | Named session |
| `tmux ls` | List sessions |
| `tmux attach -t name` | Attach to session |
| `tmux kill-session -t name` | Kill session |

---

## Sessions & windows

| Bind | Action |
|---|---|
| <kbd>prefix</kbd> <kbd>$</kbd> | Rename session |
| <kbd>prefix</kbd> <kbd>s</kbd> | List / switch sessions |
| <kbd>prefix</kbd> <kbd>c</kbd> | New window |
| <kbd>prefix</kbd> <kbd>1</kbd>…<kbd>9</kbd> | Jump to window N |
| <kbd>prefix</kbd> <kbd>Tab</kbd> | Last window |
| <kbd>prefix</kbd> <kbd>w</kbd> | Visual window/session tree |
| <kbd>prefix</kbd> <kbd>,</kbd> | Rename window |

---

## Panes

| Bind | Action |
|---|---|
| <kbd>prefix</kbd> <kbd>\|</kbd> | Split right (current path) |
| <kbd>prefix</kbd> <kbd>-</kbd> | Split down (current path) |
| <kbd>prefix</kbd> <kbd>h</kbd><kbd>j</kbd><kbd>k</kbd><kbd>l</kbd> | Navigate panes (vi) |
| <kbd>prefix</kbd> <kbd>H</kbd><kbd>J</kbd><kbd>K</kbd><kbd>L</kbd> | Resize pane (repeatable) |
| <kbd>prefix</kbd> <kbd>z</kbd> | Toggle zoom (fullscreen pane) |
| <kbd>prefix</kbd> <kbd>x</kbd> | Kill pane |
| <kbd>prefix</kbd> <kbd>q</kbd> | Show pane numbers |

---

## Copy / paste (vi mode)

1. <kbd>prefix</kbd> <kbd>[</kbd> — enter copy mode
2. Navigate with vi keys (<kbd>h</kbd><kbd>j</kbd><kbd>k</kbd><kbd>l</kbd>, <kbd>/</kbd> to search)
3. <kbd>v</kbd> — begin selection
4. <kbd>y</kbd> — yank & exit copy mode
5. <kbd>prefix</kbd> <kbd>]</kbd> — paste buffer

View buffer: `tmux show-buffer` · List all: `tmux list-buffers`

## Custom macros

| Bind | Action |
|---|---|
| <kbd>prefix</kbd> <kbd>Alt</kbd>+<kbd>3</kbd> | 3 equal columns (triple split) |
| <kbd>prefix</kbd> <kbd>r</kbd> | Reload `~/.tmux.conf` |
